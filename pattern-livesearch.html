Live Search

Default

<div class="pat-livesearch"
  data-pat-livesearch="url: /search.html">
  <input type="text" class="pat-livesearch-input"
    placeholder="Search" />
  <div class="pat-livesearch-container">
    <div class="pat-livesearch-results">
    </div>
  </div>
</div>

Data Attributes

Attribute Default Description
url The url that will be of the server side page that will be queried.
delay 400 Delay in ms after user types before searching.
param SearchableText The query string parameter which will be set with the value of the input.
highlight pat-livesearch-highlight CSS class that will be used to highlight items on keyboard navigation.
chars 3 Number of characters user must type before search begins.
input .pat-livesearch-input Selector for element that will be used for the livesearch input.
toggle-target .pat-livesearch-container Selector for element that will be shown when results are found.
toggle-klass show CSS class that will be added to target toggle element when results are found.
results-target .pat-livesearch-results Selector for element where results from the server will be inserted.
results-content Selector for element inside in the HTML from the server which contains the results.
